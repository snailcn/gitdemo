git 学习笔记
1、git init Git初始化
2、git status 查看当前git管理文件状态
3、文件状态的三种颜色
	红色：文件变更了（新增/修改） --> git add 文件名 进单个点击  或者 git add . 进行当前目录所有文件批量添加 之后会变成绿色
	绿色：代表文件已经管理起来了  --> git commit -m '版本提交的描述信息'  生成版本
	生成版本
